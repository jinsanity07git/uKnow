Ongoing project: The Next Generation Transportation Management System

Previous Next

The proposed system is a next-generation transportation management system for the predictive planning decision. It consists of four main components which include sensible network infrastructure, capable computing power, impressive visual guidance as well as robust simulation engines. Aside from inherent features such as high performance, interactivity, and usability, the whole system is deployed on the cloud and supported by a mother platform (the Utopia Platform) which ensures its compatibility and extensibility. Overall, the system framework is specifically designed to meet the large-scale management demand in the area of big data and artificial intelligence.

The department of transportation at all levels in those smart cities definitely needs this system to help them make a decision no matter in the U.S. or other countries.

Core features

Different from current one-time simulation solution which only can simulate the small or medium size street network. The system is capable of solving large scale network in a responsive method by adopting parallel simulation. Besides, simulation engines are deployed and controlled on cloud which ensures its scalability.

The simulation results are no longer a fetid and wordy report. All related information could be integrated into the Interactive operational dashboard centered on the advanced data map. Users can learn their interested information through an interactive way on a web application.

In terms of usability, the nice thing is that if something changes for instance, multiple events happen at the same time in one intersection, technicians do need to input and run the simulation for multiple times. We provide the complex event modeling ability for them to efficiently build and modify a set of scenarios in the advanced data map and run the simulation with ease.

The web-based approach provides a simple solution to support cross-platform deployment and enable easy sharing and communication between collaborators and audiences. The service is upgradable by assembling extensions from the Utopia platform as system components.
